1. CSS Unit Absolute
  - cm (centimeters)
  - mm (milimeters)
  - in (inchies)
  - px (pixels) 1px = 1/96th of 1in
  - pt (points) 1pt = 1/72 of 1in
  - pc (picas picas) 1pc = 12pt

2. CSS Unit Relative
  - % = to parent element
  - em = to Font size parent element
  - rem = ro font size root element
  - vw = to 1% of viewport width 
  - vh = to 1% of viewport height
