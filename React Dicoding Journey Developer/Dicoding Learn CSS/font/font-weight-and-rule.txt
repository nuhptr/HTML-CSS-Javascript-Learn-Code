1. Font weight
   Nilai dari properti ini dapat ditentukan menggunakan numeric value (100 - 900) atau dengan
   menggunakan descriptive terms (normal, bold, bolder, dan lighter).

2. Font style
   ada 3 yaitu normal, italic, oblique (teks normal yang dibuat miring).

3. Font variant
   membuat teks menjadi kapital dengan ukuran kecil dengan value (small-caps).
